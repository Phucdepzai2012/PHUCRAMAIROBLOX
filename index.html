📦 HƯỚNG DẪN CÀI PLUGIN ROBLOX
1
Mở Roblox Studio
Mở Roblox Studio và vào project của bạn.
2
Tạo Plugin Script
Trong ServerStorage, tạo một ModuleScript tên RobloxAIPlugin. Sau đó vào Plugins → Plugin Manager để tạo plugin mới, hoặc dùng Script dưới đây trực tiếp trong StarterPlayerScripts để test:
LUA — Plugin Code
-- RobloxAI Studio Plugin v2.0 -- Paste vào Plugin Script trong Roblox Studio local HttpService = game:GetService("HttpService") local Selection = game:GetService("Selection") local RunService = game:GetService("RunService") -- ⚠️ THAY USER_ID BẰNG ID CỦA BẠN (lấy từ web app) local USER_ID = "PASTE_YOUR_USER_ID_HERE" local FIREBASE_URL = "https://vietnam-life-game-default-rtdb.asia-southeast1.firebasedatabase.app" local POLL_INTERVAL = 2 -- giây local function getCommands() local ok, result = pcall(function() local url = FIREBASE_URL .. "/commands/" .. USER_ID .. "/queue.json" local res = HttpService:GetAsync(url, true) return HttpService:JSONDecode(res) end) if ok then return result else return nil end end local function markDone(cmdId) pcall(function() local url = FIREBASE_URL .. "/commands/" .. USER_ID .. "/queue/" .. cmdId .. ".json" HttpService:PatchAsync(url, HttpService:JSONEncode({status = "done"})) end) end local function executeCommand(cmdId, cmd) if cmd.type == "script" then -- Chạy script Lua local ok, err = pcall(function() local fn = loadstring(cmd.content) if fn then fn() end end) if not ok then warn("[RobloxAI] Script error: " .. tostring(err)) end elseif cmd.type == "model" then -- Tạo model từ mô tả (thực thi code tạo part/model) local ok, err = pcall(function() local fn = loadstring(cmd.content) if fn then fn() end end) if not ok then warn("[RobloxAI] Model error: " .. tostring(err)) end elseif cmd.type == "chat" then -- Hiển thị tin nhắn trong output print("[RobloxAI] " .. tostring(cmd.content)) end markDone(cmdId) end -- Main polling loop print("[RobloxAI] ✅ Plugin đã khởi động! User: " .. USER_ID) while true do local commands = getCommands() if commands then for cmdId, cmd in pairs(commands) do if cmd.status == "pending" then print("[RobloxAI] ⚡ Thực thi lệnh: " .. tostring(cmd.type)) executeCommand(cmdId, cmd) end end end wait(POLL_INTERVAL) end
3
Dán User ID
Copy User ID từ sidebar bên trái và thay vào dòng USER_ID = "PASTE_YOUR_USER_ID_HERE" trong plugin.
4
Bật HTTP Requests
Vào Game Settings → Security và bật "Allow HTTP Requests" để plugin có thể kết nối Firebase.
5
Bắt đầu sử dụng!
Chạy game trong Studio, rồi quay lại web app và ra lệnh cho AI. Code sẽ tự động thực thi trong Roblox Studio của bạn! 🚀
⚠️ Lưu ý bảo mật: Chỉ chạy plugin trong project Roblox Studio cá nhân của bạn. Không chia sẻ User ID với người khác.
