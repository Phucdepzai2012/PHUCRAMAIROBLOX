📦 HƯỚNG DẪN PLUGIN ROBLOX
1
Tải Plugin Script
Nhấn "⬇️ Tải Plugin .lua" ở sidebar trái — file đã có sẵn User ID của bạn.
2
Tạo Plugin trong Studio
Mở Roblox Studio → Plugins → Plugin Manager → Create New Plugin → Paste nội dung .lua vào.
3
Bật HTTP Requests
Game Settings → Security → Allow HTTP Requests: ON
4
Chạy và ra lệnh!
Nhấn Play → quay lại web, chat với AI → nhấn 🚀 → Roblox trên code block để thực thi ngay trong Studio.
⚠️ Chỉ dùng trong project cá nhân. Không chia sẻ User ID với người khác.
